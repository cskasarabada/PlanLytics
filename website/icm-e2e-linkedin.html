Use Cases

Built for Oracle ICM teams

1.

New Plan Implementation

Upload a compensation plan document (PDF, DOCX, or pasted text). The AI agent extracts plan structure, maps to Oracle ICM objects, generates all configuration artifacts, and optionally deploys via REST API.

Input: Plan document → Output: Live ICM configuration
2.

Year-Over-Year Plan Migration

Import the current year's IcCnPlanCopy XML export. Migrate to the next plan year with updated date ranges, year-suffix logic, and credit category reuse. Export the new XML for upload.

Input: 2025 XML → Output: 2026 XML + workbook
3.

Configuration Health Check

Import an existing Oracle ICM XML export and run the Efficiency Report. Find duplicate expressions, unused rate tables, missing goals, and consolidation candidates. Score your config 0-100.

Input: Current XML → Output: Efficiency score + findings
4.

SI Accelerator

System integrators can use ICM E2E to accelerate client engagements. Generate Design Documents and Configuration Documents from plan requirements, then deploy to the client's Oracle instance.

Input: Client requirements → Output: Full SI deliverable set
